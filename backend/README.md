# DEMO-JDBC04 Backend

Spring Boot Web, JPA, JDBC
Spring Security, JJWT
Postgresql JDBC driver, AgensGraph JDBC driver

### Prerequisites

* Java SE 8 / Java 1.8
* Maven 3

### Installing

mvn clean install -DskipTests

## Running the app

mvn spring-boot:run

## Built With

* [Spring Boot](https://projects.spring.io/spring-boot/)
* [Maven](https://maven.apache.org/)
